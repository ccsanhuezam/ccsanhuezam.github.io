FOTOGRAMETRÍA CON RECAP

Renders

Aquí se mostrará un poco lo que fue hacer fotogrametría con el programa Recap Photo de un plato de frutas, en este caso, plátanos, kiwi y granada. Para esto se presentarán distintos renders, con fondo plano, de detalle y con contexto, elaborados con la aplicación Keyshot. Finalmente se presentará un video del fotograma para así mostrarlo en 360º.
